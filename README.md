# Project 3: Spots

## Description

Spots is a new innovative way to share art, images and find inspiration. Each user will have the ability to create their own Spot on the website and upload pictures of their choosing. Visitors to the Spots page will have the ability to hit like to show their support.

## Tech Stack

- HTML
- CSS
- Responsive Design

## Deployment

This webpage is deployed to Github pages.

## Plan on improving the project

- Display select social media entries.
- More customizible environments. User can change background image, possible area for AI image generation integration.
- Possible 3D customizible environments where images are placed in a gallery style. Adapt view and navigation for AR/VR.
- User to user messaging.
